# EcoRota

Aplicativo mobile de rotas ecológicas, com backend hospedado online.

---

## Sobre o Projeto

O projeto **EcoRota** foi desenvolvido por **Naiade Alves, Joana Marques e Carla Daniela** para a disciplina **Tópicos Integradores** do curso de **Análise e Desenvolvimento de Sistemas**.  
O aplicativo permite que os usuários encontrem pontos de coleta para reciclagem de diversos materiais, visualizem rotas e localização no mapa, acompanhem suas recompensas por materiais reciclados e gerenciem seu perfil individual.

---

## Tecnologias Utilizadas

- **Frontend:** React Native com Expo  
- **Backend:** Node.js, Express  
- **Banco de Dados:** MongoDB Atlas
